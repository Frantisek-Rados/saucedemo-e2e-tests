# Saucedemo E2E testy

## Cieľ
Otestovať celý nákupný proces na SauceDemo - od prihlásenia po odhlásenie.

## Testované scenáre
1. Prihlásenie (úspešné, neúspešné, prázdne polia)
2. Pridanie produktu do košíka
3. Odstránenie produktu z košíka
4. Dokončenie objednávky (vyplnenie údajov, potvrdenie)
5. Odhlásenie

## Nástroje
- Python + pytest
- Playwright (UI)
- Page Object Model (POM)
- Github Actions (CI/CD)
